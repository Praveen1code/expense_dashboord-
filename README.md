💰 Expense Dashboard

A real-time collaborative expense tracking application where multiple users can add, edit, and manage monthly expenses together. All updates are synchronized instantly across connected users.

✨ Features
- 📊 Real-time expense tracking
- 👥 Multi-user collaboration
- 💸 Add, edit, and delete expenses
- 📅 Monthly expense summary
- 📈 Category-wise analytics
- 📱 Responsive design
